Panduan RiveScript

Apa itu RiveScript?

RiveScript adalah bahasa scripting sederhana untuk membuat chatbot. Kamu bisa mengatur apa yang user ketik dan apa yang bot jawab.

1. Struktur Dasar

+ hello
- Hello!

Artinya: jika user mengetik hello, bot menjawab Hello!

2. Simbol Penting

  • + untuk input dari user
  • - untuk jawaban bot
  • * untuk wildcard
  • ! untuk pengaturan
  • > dan < untuk topic/grouping

3. Contoh Chatbot Sederhana

! version = 2.0

+ hello
- Hello there!

+ what is your name
- My name is Bot.

+ how are you
- I'm fine!

+ *
- Sorry, I don't understand.

4. Menggunakan Wildcard

+ my name is *
- Nice to meet you, <star>!

Jika user mengetik my name is John, bot menjawab Nice to meet you, John!

5. Menyimpan Data

+ my name is *
- <set name=<star>>Nice to meet you!

+ what is my name
- Your name is <get name>.

Bot bisa mengingat nama user dengan variable.

6. Kondisi

+ i am # years old
- <set age=<star>>Got it.

+ am i adult
* <get age> >= 18 => Yes, you are an adult.
- No, you are underage.

7. Topic

> topic food

+ i like pizza
- Pizza is delicious!

+ i like burger
- Burger is great too!

< topic

8. Redirect

+ hi
@ hello

+ hello
- Hello there!

Simbol @ dipakai untuk redirect ke trigger lain.
